What is Global Azure Bootcamp?

On, Saturday, April 16, 2016  we are out to set some records again!                      

In April of 2013 the first Global Windows Azure Bootcamp  was held at more than 90 locations around the globe! In March 2014 it was topped with 136 locations, and then again last year in 2015 we hit 182 locations!  This year we are again doing a one day deep dive class to help thousands of people get up to speed on developing Cloud Computing Applications for Azure .

In addition to this great learning opportunity we will have another set of hands on labs

Microsoft Dev Community Odisha is hosting Azure Bootcamp this year, we have sessions on fascinating topics like Azure Websites, Azure Storage, Mobile Services and IoT.
