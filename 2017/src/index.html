What is Global Azure Bootcamp?

On, Saturday, April 22, 2017  we are out to set some records again!                      

Global Azure Bootcamp (#globalazure) is turning five years old and is a community owned and community driven day of Azure all around the world. On the same day, in 2017 that day is April 22, hundreds of user groups and tech communities run a one day, free to attend, meetup (or bootcamp) where we learn together about Azure. Each UG will arrange the agenda for the day, and with the content that makes sense, for them. Globally we arrange for sponsors with nice giveaways. What makes the Global version different is, well, it’s global! In 2016 over 187 locations around the world hosted a bootcamp on the same day. It’s been growing each year since and at writing moment we have 230 locations signed up for the 2017 event!

Microsoft Dev Community Odisha is hosting Azure Bootcamp this year, we have sessions on fascinating topics like Azure Websites, Azure Storage, Mobile Services and IoT.
