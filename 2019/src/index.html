What is Global Azure Bootcamp?

On, Saturday, April 27, 2019  we are out to set some records again!                      

Welcome to Global Azure Bootcamp! All around the world user groups and communities want to learn about Azure and Cloud Computing! On April 27, 2019, all communities will come together once again in the sixth great Global Azure Bootcamp event! Each user group will organize their own one day deep dive class on Azure the way they see fit and how it works for their members. The result is that thousands of people get to learn about Azure and join together online under the social hashtag #GlobalAzure!
Utkal Techies Forum, Odisha is hosting Azure Bootcamp this year, we have sessions on fascinating topics like Azure Websites, Azure Storage, Mobile Services and IoT.
