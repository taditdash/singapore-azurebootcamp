What is Global Azure Bootcamp?

On, Saturday, April 25, 2015  we are out to set some records again!                      

In April of 2013 the first Global Windows Azure Bootcamp  was held at more than 90 locations around the globe! In March 2014 it was topped with 136 locations!  This year we are again doing a one day deep dive class to help thousands of people get up to speed on developing Cloud Computing Applications for Azure .

In addition to this great learning opportunity we will have another set of hands on labs

Microsoft Dev Community Odisha is hosting Azure Bootcamp this year, we have sessions on fascinating topics like Azure Websites, Azure Storage, Mobile Services and IoT.



MEET OUR SPEAKERS




Abhishek Sur
India

Mr. Abhishek Sur, who is a well known and popular Author, Public Speaker and Blogger. He is a Microsoft MVP, Code Project MVP, Dotnetfunda MVP. Abhishek authored one of the best selling book of .NET. It covers ASP.NET, WPF, Windows 8, Threading, Memory Management, Internals, Visual Studio, HTML5, JQuery and many more... Another book on .NET 4.5 has been released very recently. It covers Debugging, Testing, Extensibility, WCF, Windows Phone, Windows Azure and many more...


Nirmal Hota
India

Nirmal Hota is a Software Developer, Tech lead, Blogger, Tech Community Speaker. He is working in Mindfire Solutions as a Tech Lead. He is a Microsoft Certified Technology Specialist in ASP.net. He is also a Microsoft Certified Dynamics CRM Consultant in Dynamics CRM app development, customization and configuration. He is also a Titanium Certified Application Developer in mobile app development. Recently, he has been awarded as the Most Valuable Blogger by DZone. Know more about Nirmal from his blog http://www.nirmalhota.com.


Tadit Dash
India

Tadit Dash is a Software Engineer at Mindfire Solutions, India. He is the first Microsoft MVP of Odisha, India. The Code Project Awarded him MVP as well and DZone has recognized him as Most Valuable Blogger. He blogs at http://taditdash.wordpress.com/. He is a die hard fan of .NET Technologies. He enjoys working on ASP.NET, JavaScript, jQuery, Dynamics CRM, Knockout.js, Angular.js.


Ankit Mohanty
India

Ankit Mohanty is an Undergraduate at KIIT University, Bhubaneswar. He is the first Xamarin Student Ambassador from Odisha. He is the coordinator of Konnexions the I.T. Society of KIIT University. Microsoft India has recognized him as a distinguished member of the App Builder's Club. He has also won the Publish event of Microsoft. He is a die hard fan of Windows Phone and loves evangelizing on Windows platform.
